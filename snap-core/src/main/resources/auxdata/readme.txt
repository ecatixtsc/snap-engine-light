
This is the directory $BEAM_HOME/auxdata

It stores auxiliary data used by the BEAM software:
  $BEAM_HOME/auxdata/rgb_profiles  - predefined RGB profiles
  $BEAM_HOME/auxdata/color_palette - predefined color palettes for VISAT's contrast stretching / color palette tool
  $BEAM_HOME/auxdata/dem           - digital elevation models
  $BEAM_HOME/auxdata/sst           - auxiliary data for the AATSR Sea Surface Temperature Processor
  $BEAM_HOME/auxdata/smile         - auxiliary data for the MERIS Smile Correction Processor
  $BEAM_HOME/auxdata/smac          - auxiliary data for the MERIS/AATSR SMAC Processor
  $BEAM_HOME/auxdata/smac          - auxiliary data for the MERIS Cloud Probability Processor
