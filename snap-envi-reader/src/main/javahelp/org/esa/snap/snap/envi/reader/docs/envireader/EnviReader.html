Envi Data Product Reader

Envi Reader

The reader supports data products in the Envi data format. The products consist of pairs of a header (.hdr) and an image file. The image file must have the same file name as the the header file. As extension the following extensions are tested in the given order(.dat, ' ',.img, .bin, .bip, .bil or .bsq). The file which is first found is used.

The data can also be read from a zip file. The first header file which can be found within the zip file is used.

HICO Reader

The data of the HICO sensor (http://www.sdl.usu.edu/programs/hico) is delivered in the Envi format. Beside the header and image file HICO products come along with other ancillary files like RGB, NDVI and flag file.
